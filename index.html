Modalify.js Demo and Documentation

1. Alert Types

These are the default alert types available:


// Example: Default Alert
showModalAlert({
  type: 'default',
  headline: 'Default Alert',
  description: 'This is a default alert example.',
  actions: [{ label: 'OK', handler: () => alert('OK clicked') }]
});

2. Alert with Multiple Actions

Alerts can have multiple actions:


// Example: Alert with Multiple Actions
showModalAlert({
  type: 'default',
  headline: 'Choose an Action',
  description: 'Please select one of the available options.',
  actions: [
    { label: 'Option 1', handler: () => alert('Option 1 clicked') },
    { label: 'Option 2', handler: () => alert('Option 2 clicked') },
    { label: 'Cancel', handler: () => alert('Cancel clicked') }
  ]
});

3. Dismissible Alerts

Alerts can be dismissed by clicking outside the modal:


// Example: Dismissible Alert
showModalAlert({
  type: 'alert',
  headline: 'Dismissible Alert',
  description: 'Click outside the modal to dismiss this alert.',
  actions: [{ label: 'OK', handler: () => alert('OK clicked') }],
  dismissible: true
});

4. Alert with Long Description

This example demonstrates an alert with a longer description:


// Example: Alert with Long Description
showModalAlert({
  type: 'error',
  headline: 'Error Alert',
  description: 'An unexpected error occurred while processing your request. ' +
    'Please check your internet connection and try again. If the problem persists, ' +
    'contact support for further assistance. We apologize for the inconvenience caused.',
  actions: [{ label: 'Close', handler: () => alert('Closed') }]
});

5. Instructions to Use

To use the modal alert system, include the modalAlert.js script in your HTML:

<script src="modalAlert.js"></script>

Call the showModalAlert function with the desired parameters:


showModalAlert({
  type: 'default', // Options: 'default', 'alert', 'error'
  headline: 'Your Title Here',
  description: 'Your description here.',
  actions: [
    { label: 'OK', handler: () => alert('OK clicked') }
  ],
  dismissible: false // Optional: Set to true to allow outside click dismissal
});
